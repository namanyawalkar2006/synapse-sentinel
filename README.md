## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
# 🛡️ Synapse Sentinel
> **The Forensic Autopsy HUD for Autonomous AI Systems**

**Synapse Sentinel** is a high-stakes investigator dashboard built to solve the "Black Box" problem of modern AI. As agents move from simple chat to autonomous decision-making, we need more than just logs—we need a real-time integrity engine that identifies logic drift and repairs ethical runoff before it’s too late.

---

## 🚀 The Vision
Most AI safety tools tell you *that* a model failed. **Synapse Sentinel** tells you *where* it failed, *why* it drifted, and generates an immediate **Ethical Patch** to re-align the system. 

Built for the **Synapse Innovation Hack 2026**, this project showcases a future where AI is accountable, transparent, and auditable.

## ✨ Core Innovations
* **Forensic "Fracture Point" Detection:** We pinpoint the exact moment an LLM bypasses its safety heuristics.
* **Dynamic Risk Scoring:** Real-time analytics for Logic Drift, Hallucination Probability, and Ethical Runoff.
* **Adaptive Intelligence:** Leveraging **Gemini 1.5 Pro** to analyze failures and generate corrected System Directives in real-time.
* **Cyber-Diagnostic UI:** A high-intensity, immersive HUD built for rapid response and forensic investigation.

## 🧰 The Tech Stack
* **Intelligence Engine:** [Google Gemini 1.5 Pro](https://aistudio.google.com/) (Root-Cause Analysis & Logic Repair).
* **Frontend Architecture:** Next.js 14 & Tailwind CSS (Cyberpunk-Professional Aesthetic).
* **Motion & Interaction:** Framer Motion & Lucide React (System Glitch & Scanline Effects).
